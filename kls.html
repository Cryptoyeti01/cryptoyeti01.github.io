Helpful tips for those trying to get KLS Karlsen coin going.

Setting up wallet and obtaing a KLS address

  1. Download the client: https://github.com/karlsen-network/karlsend/releases
  2. Unzip files and put somewhere you can easily find them. I just create a dedicated directory like C:\Karlsen and place them all in there.
  3. Open up a command prompt or powershell and change to the Karlsen directory (i.e. cd C:\Karlsen).
    Type in the command: karlsend.exe --utxoindex
    This will start the node and sync it to the blockchain. Note that it may take some time to fully sync up with the network.
  4. Open up a new command prompt or powershell (leave previous window open) and change to the Karlsen directory (i.e. cd C:\Karlsen).
    Type in the command: karlsenwallet.exe start-daemon
    This will start the wallet daemon in order for you to execute the next steps.

  5. Create a new wallet


    Type in the command: karlsenwallet.exe create
    This will create a new wallet keyfile.

  6. Reveal and save your seed words


    Type in the command: karlsenwallet.exe dump-unencrypted-data
    This will display your newly created wallets seed words. Write these down and keep them safe.
    These words can be used to recreate your wallet if it becomes corrupted. Be careful as anyone who has these words can also gain access to your coins.

  7. Create a new receiving address


    Type in the command: karlsenwallet.exe new-address
    This will create a new wallet receiving address you can send KLS coins to.

  8. Check balance


    Type in the command: karlsenwallet.exe balance
    This will display the balance of the address you created above.
    Alternatively, type in the command: karlsenwallet.exe balance -v
    The addition of the -v flag will display all addresses with a balance. This is useful to also display change balances once you made a few transactions.
